# terminal-setup

Practice project: terminal-setup

## Description
This project focuses on setting up and configuring a terminal environment for various development tasks.

## Features
- Custom terminal prompts
- Aliases for common commands
- Setup scripts for quick installation

## Tech Stack
- Python
- Shell scripting

## Installation
To install the configuration, run:
```bash
bash install.sh
```

## Usage
To use the setup, open your terminal and execute the following command:
```bash
source ~/.bashrc
```
